Universal Card System (Implementado)

lightbulb

Universal Card Básica

Sistema unificado que reemplaza simple-card, standard-card y complementary-card.

  • Alineación izquierda consistente
  • Hover effects suaves
  • Sistema de checkmarks

Universal Card Extensiva

Modificador .extensive para contenido narrativo extenso. Incluye padding aumentado y texto justificado para mejor legibilidad en contenido largo.

Ideal para misión, visión, textos académicos y contenido educativo detallado. Optimizada para párrafos múltiples.

settings

Universal Card Técnica

Modificador .technical para documentación técnica y especificaciones.

Incluye tamaño de título aumentado y espaciado optimizado para contenido estructurado.
